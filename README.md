# Riss_Popular_thesis_wordCloud

Riss 에서 제공하는 인기 학술 자료의 데이터를 사용하였다.  
http://www.riss.kr/PopularList.do  

상단의 기간 변경을 기준으로 데이터를 추출하는 것으로 진행중이며  
해당 페이지의 99개의 자료를 크롤링 하여 텍스트 파일로 저장,    
저장된 텍스트 파일을 가공하여 워드클라우드로 생성한다.
